## Что такое Контур Фокус

**Контур Фокус** — сервис Контура для оценки надёжности контрагентов. Позволяет проверять организации перед сделкой: ликвидация, банкротство, исполнительные производства, финансовая активность и другие риски.

Подробнее: [Контур.Помощь — Оценка надёжности контрагента (Фокус)](https://support.kontur.ru/focus-tm/51373-ocenka_nadezhnosti_kontragenta).

## Методы оценки

### Рейтинг надёжности (маркеры)

- Условная оценка по шкале **0–100** на основе более чем 80 маркеров (красные, жёлтые, зелёные).
- Учитываются: ликвидация/банкротство, исполнительные производства, адреса массовой регистрации, финансовые показатели, госконтракты, активность компании и др.

**Обозначения:**

| Значок   | Рейтинг / условие |
|----------|--------------------|
| Зелёный  | 70–100             |
| Серый    | 70 при отсутствии сработавших маркеров |
| Жёлтый   | 40–69 при наличии жёлтых маркеров      |
| Красный  | 0–39 при наличии красных маркеров      |

### Проверка «Светофор»

- Экспресс-метод без ручной настройки критериев.
- Факты о контрагенте делятся на категории: **нейтральная** (существенных фактов нет), **зелёная** (признаки активности), **жёлтая** (факты, требующие внимания).

Расчёт рейтинга может отличаться в зависимости от тарифного плана: «Универсальный», «Оптимальный», «Базовый».

## Где использовать

- Перед заключением договора с новым контрагентом.
- При регулярном мониторинге уже работающих контрагентов.
- В 1С: интеграция «Фокус: 1С» позволяет проверять контрагентов из карточки организации. Справка: [Настройка доступа к списку (Фокус: 1С)](https://support.kontur.ru/focus-1s8x/42295-nastrojka_dostupa_k_spisku_drugim_polzovatelyam).
