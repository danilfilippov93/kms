# База знаний (KMS)

Локальная база знаний в формате Markdown по сценарию на Git.

- **Клиентский сервис** — инструкции для поддержки.
- **Продажи** — инструкции для отдела продаж.

Статьи можно открыть через меню слева. Редактирование — в файлах `content/` в репозитории.
