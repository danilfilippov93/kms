# База знаний (KMS) на Git

Минимальная база знаний по сценарию из документа «KMS-тезисы-технической-реализации.md»: контент в Markdown с YAML frontmatter, просмотр через MkDocs Material.

## Структура

- **content/** — статьи: `content/категория/локаль/статья-id.md`
- **KMS-тезисы-технической-реализации.md** — тезисы для технической реализации
- **Инструкция-репозиторий-и-запуск.md** — как создать репозиторий на GitHub и запустить просмотр локально
- **mkdocs.yml** — конфиг MkDocs для сборки сайта

## Быстрый старт

1. Установить MkDocs и тему: `pip install mkdocs mkdocs-material`
2. Запустить просмотр: `mkdocs serve`
3. Открыть в браузере: http://127.0.0.1:8000

Редактирование — в любом Markdown-редакторе (Cursor, VS Code) или на GitHub в браузере.
